# UniSP Download Page

Static download landing page (no build step).

## Run locally

- Open `index.html` directly in your browser, or serve the folder:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Customize

- Edit `index.html` to change app name, version, and download links.
- Replace `REPLACE_WITH_SHA256` with real hashes (optional).
- Put your installers in `downloads/` (or point links to your CDN / GitHub Releases).
